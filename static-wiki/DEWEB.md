# DEWEB (Development for Web & Mobile)
This folder contains all the assets from the DEWEB module. It was a module to get to know JavaScript and web development in general. We had to either code a soccer game or a interactive book using HTML, CSS and Vanilla JavaScript.

## Projektarbeit
### ✨ The art of Nika ✨
✨ The art of Nika ✨ ist ein Projekt, das auf reinem HTML, CSS und Vanilla Javascript läuft. Es ist ein interaktives, animiertes Bilderbuch, in dem die Buchstaben des Vornamens auf den einzelnen Seiten animiert sind.

Das ganze Projekt ist responsive, d.h. der Inhalt passt sich der Bildschirmgrösse an. 

Auf Desktop wird ein kleines Game gestartet, bevor der eigentliche Content angezeigt wird. Das Minigame ist mit Sound ausgestattet und das Spritesheet des Players ist selber gestaltet.
